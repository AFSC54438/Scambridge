3. Open **Google Chrome**  
4. In the address bar, go to `chrome://extensions/` and press **Enter**  
5. Enable **Developer mode** using the toggle in the top-right corner  
6. Click **Load unpacked**  
7. Select the **extracted Booklet folder** (the folder containing `manifest.json`)  
8. Confirm the extension appears in the extensions list  
9. (Optional) Pin the extension by clicking the **puzzle icon** (ðŸ§©) in the toolbar and selecting the **pin** icon next to **Booklet**



## Updating the Extension

1. Replace the existing Booklet folder with the new version  
2. Open `chrome://extensions/`  
3. Click **Reload** on the Booklet extension
